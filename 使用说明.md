# 交安眼离线平台使用说明

打开方式：

```text
C:\Users\ASUS\Desktop\AI视觉组\JiaoAnYan-platform\index.html
```

使用流程：

1. 点击左侧“导入交通图片”，选择 JPG / PNG / WebP 图片。
2. 在“应用场景”中选择车载预警、流量监控或道路巡检。
3. 按需打开或关闭光照校正、频域感知增强、边缘/病害提取、候选目标框、轨迹与风险叠加。
4. 点击“处理图片”重新生成结果。
5. 点击“导出结果”保存处理后的 PNG 图片。

说明：

- 这是纯本地离线平台，不会上传图片。
- 当前版本使用 Canvas 图像处理与启发式候选框模拟 MCWTT 的输出链路。
- 后续如果要接入真正训练好的 CSWinTT / MCWTT 模型，可以把模型推理结果接到 `app.js` 的候选框、置信度和事件输出位置。
